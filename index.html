We can't fix what we can't measure.

We can use already public data to understand the police systems around us. Through direct action, we can cover each agency and paint a national picture of the state of policing.

Community

We're an open source learning organization. Talk to us in Discord and GitHub.

Data Collection

Our goal is to support volunteer Data Scrapers as they write Python scripts to collect data from websites.

There are about 18,000 police organizations, and each has a unique way to make data public. We can make it usable, measurable, and actionable by putting it in one place.

Contribute code

Data Storage

We store the data to archival standards, and facilitate open source translation and analytics. Our data and source scrapers are publicly available, free of charge.
